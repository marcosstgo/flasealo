# ✅ Checklist Rápido Pre-Festival (15 minutos)

## 🔥 CRÍTICO - Hacer AHORA

### 1. Crear Evento de Prueba (5 min)
```
□ Login como organizador
□ Crear evento: "Test Festival [Fecha]"
□ Activar "Permitir descargas"
□ Marcar como "Público"
□ Generar código QR
□ IMPRIMIR o GUARDAR QR en tu teléfono
```

### 2. Probar Flujo Completo (5 min)
```
□ Escanear QR desde tu teléfono
□ Ingresar nombre de prueba
□ Aceptar consentimiento
□ Subir 2-3 fotos de prueba
□ Verificar que aparezcan en Dashboard
□ Aprobar las fotos
□ Verificar que aparezcan en Galería pública
□ Probar descargar fotos
```

### 3. Configurar Supabase Dashboard (5 min)
```
□ Ir a: dashboard.supabase.com
□ Storage → event-photos → Settings:
   ☑ File size limit: 10 MB
   ☑ Allowed MIME: image/jpeg,image/png,image/jpg
   ☑ Public: ✅
□ Authentication → Email Auth:
   ☑ OTP expiry: 900 seconds (15 min)
   ☑ Enable Password Protection: ✅
```

## 🚨 PLAN DE CONTINGENCIA

### Si falla la conexión:
```
□ Opción 1: Usar hotspot 4G del teléfono
□ Opción 2: Postponer subidas hasta tener WiFi
□ Opción 3: Usar app de backup (Google Photos compartido)
```

### Si Supabase tiene problemas:
```
□ Verificar: status.supabase.com
□ Si es caída general: esperar (generalmente < 15 min)
□ Si es problema de cuenta: contactar soporte
```

### Si la galería no carga:
```
□ Verificar desde otro dispositivo
□ Limpiar cache del navegador
□ Usar modo incógnito
□ Verificar estado de Storage en Supabase
```
